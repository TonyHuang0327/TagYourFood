# TagYourFood

## 簡介
TagYourFood 是一個旨在幫助用戶標記和管理食物的應用程式。通過這個應用程式，用戶可以輕鬆地記錄和追踪他們的飲食習慣，並獲得健康建議。

## 功能
- **食物標記**：快速標記和分類食物。
- **飲食追踪**：記錄每日飲食，追踪營養攝入。
- **健康建議**：基於用戶的飲食數據提供健康建議。
- **數據可視化**：通過圖表展示飲食數據。
